
# Roadmap

## MVP (0.1.x)
- Notebook editor + runner (ipynb)
- Offline scientific stack: numpy/pandas/matplotlib
- Streamlit runner + WebView preview
- GitHub login + clone/pull/push
- Controlled package installation (safe mode)

## v0.2
- Better notebook outputs (HTML, richer tables)
- Interrupt execution improvements
- Templates gallery (notebook + streamlit starters)

## v0.3
- Optional additional ABIs
- On-demand python-pack delivery (PAD)
- Performance profiling and cold-start improvements
